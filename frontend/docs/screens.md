# 画面仕様書

## 概要

PodLog フロントエンドの全画面仕様を定義する。各画面について目的、URL、表示要素、ユーザー操作、使用 API、状態別表示を記載する。

---

## 画面一覧

| # | 画面名 | URL | 認証 | レイアウト |
|---|--------|-----|------|-----------|
| 1 | ランディングページ | `/` | 不要 | 独自 |
| 2 | サインアップ | `/signup` | 不要 | auth |
| 3 | ログイン | `/login` | 不要 | auth |
| 4 | プロフィール初期設定 | `/profile/setup` | 必要 | main |
| 5 | ダッシュボード（検索） | `/dashboard` | 必要 | main |
| 6 | ポッドキャスト詳細 | `/podcasts/[id]` | 必要 | main |
| 7 | エピソード詳細 | `/episodes/[id]` | 必要 | main |
| 8 | タイムライン | `/timeline` | 必要 | main |
| 9 | マイプロフィール | `/profile` | 必要 | main |
| 10 | ユーザー公開プロフィール | `/users/[username]` | 必要 | main |
| 11 | 設定 | `/settings` | 必要 | main |

### レイアウト種別

- **独自**: Navbar なし。ページ固有のヘッダー・フッターを持つ
- **auth**: Navbar なし。認証フォーム用の中央寄せレイアウト
- **main**: 上部に Navbar を表示。コンテンツは `max-w-4xl` で中央寄せ、上下パディングあり

## 共通コンポーネント: Navbar

### デスクトップ（sm 以上）

```
┌──────────────────────────────────────────────┐
│  🎙 PodLog          検索       [Avatar] 名前  │
└──────────────────────────────────────────────┘
```

- **ロゴ（PodLog）**: `/dashboard` へ遷移
- **検索**: `/dashboard` へ遷移（現在のパスが `/dashboard` のときハイライト）
- **アバター + 名前**: 表示のみ（リンクなし）

### モバイル（sm 未満）

```
┌──────────────────────────────────────────────┐
│  🎙 PodLog                            [≡]    │
└──────────────────────────────────────────────┘
```

ハンバーガーメニュー押下でサイドバーが開く:

```
┌────────────────────┐
│  [Avatar]           │
│  表示名             │
│  @username          │
│                     │
│  検索               │  → /dashboard
│  プロフィール        │  → /profile
│  設定               │  → /settings
│                     │
│  ログアウト          │
└────────────────────┘
```

│                                              │
│         [無料で始める]  [ログイン]              │
│                                              │
├──────────────────────────────────────────────┤
│                                              │
│  ┌──────────┐ ┌──────────┐ ┌──────────┐     │
│  │ 🔍       │ │ 🎧       │ │ 👤       │     │
│  │ 番組を    │ │ エピソード │ │ プロフィール│     │

なし

## 2. サインアップ

- **URL**: `/signup`
- **認証**: 不要
- **目的**: 新規ユーザーのアカウント作成

### 表示要素

```
┌──────────────────────────────────────────────┐
│                                              │
│              PodLog に登録                    │
│                                              │
│  メールアドレス: [________________]           │
│  パスワード:     [________________]           │
│  パスワード確認: [________________]           │
│                                              │
│              [登録する]                       │
│                                              │
│    すでにアカウントをお持ちですか？ ログイン    │
│                                              │
└──────────────────────────────────────────────┘
```

### ユーザー操作

| 操作 | 挙動 |
|------|------|
| 「登録する」押下 | Supabase Auth でアカウント作成 → `/dashboard` へ遷移 |
| 「ログイン」リンク押下 | `/login` へ遷移 |

### バリデーション

- パスワード: 6文字以上
- パスワード確認: パスワードと一致すること

### 状態別表示

| 状態 | 表示 |
|------|------|
| エラー | フォーム上部に赤字でエラーメッセージ表示 |
| 送信中 | ボタンが無効化される |

### 使用 API

- Supabase `auth.signUp`

```
┌──────────────────────────────────────────────┐

| プロフィール未設定 | `/profile/setup` へリダイレクト |

### 使用 API

- `GET /api/v1/podcasts/search?q={query}`

## 6. ポッドキャスト詳細

- **URL**: `/podcasts/[id]`
- **認証**: 必要
- **目的**: ポッドキャストの詳細情報表示とエピソード一覧

### 表示要素

```
┌──────────────────────────────────────────────┐
│  [Navbar]                                    │
├──────────────────────────────────────────────┤
│                                              │
│  ← 戻る                                     │
│                                              │
│  ┌─────────────────────────────────────┐     │
│  │ [大きいArtwork]                      │     │
│  │                                     │     │
│  │ タイトル                             │     │
│  │ 著者名                               │     │
│  │ ジャンル                              │     │
│  │                                     │     │
│  │ 説明文...                            │     │
│  │                                     │     │
│  │ [iTunes で見る]                       │     │
│  └─────────────────────────────────────┘     │
│                                              │
│  ── エピソード ─────────────────────────      │
│                                              │
│  （フィード取得結果メッセージ）                  │
│  [フィードから更新]                            │
│                                              │
│  ┌────────────────────────────────────┐      │
│  │ エピソードタイトル                    │      │
│  │ 公開日 ・ 再生時間                    │      │
│  │ 説明文（2行まで）...                  │      │
│  └────────────────────────────────────┘      │
│  ...                                         │
│                                              │
│  [もっと見る]                                 │
│                                              │
└──────────────────────────────────────────────┘
```

### ユーザー操作

| 操作 | 挙動 |
|------|------|
| 「戻る」押下 | ブラウザバック |
| 「iTunes で見る」押下 | 外部リンク（iTunes）を新規タブで開く |
| 「フィードから更新」押下 | RSS フィードからエピソード取得・保存 |
| エピソードカード押下 | `/episodes/[id]` へ遷移 |
| 「もっと見る」押下 | 次のページのエピソードを読み込み |

### 自動処理

- ページ表示時に自動で RSS フィードからエピソードを取得する

### 状態別表示

| 状態 | 表示 |
|------|------|
| ローディング | Loading コンポーネント表示 |
| ポッドキャスト未発見 | 「ポッドキャストが見つかりませんでした」 |
| エピソード 0 件 | 「エピソードはまだありません」 |
| フィード取得成功 | 「N件のエピソードを取得しました（新規: M件）」 |
| フィード取得中 | ボタン無効化 + 「取得中...」 |
| エラー | ErrorMessage コンポーネント表示 |

### 使用 API

- `GET /api/v1/podcasts/{id}`
- `GET /api/v1/podcasts/{id}/episodes?limit=20&offset=0`
- `POST /api/v1/podcasts/{id}/episodes/fetch`

- **URL**: `/episodes/[id]`

- **目的**: エピソードの詳細表示、聴取記録、レビューの閲覧・投稿

│                                              │
│  ★4.2 (5件のレビュー)                         │
│                                              │
│  [🎧 聴いた] / [聴いたにする]                  │
│                                              │
│  🔊 [オーディオプレーヤー]                     │
│                                              │
│  説明文...                                    │
│                                              │
│  [配信元で聴く]                               │
│                                              │
│  ── レビュー ──────────────────────────       │
│                                              │
│  ┌────────────────────────────────────┐      │
│  │ レビューを書く                       │      │
│  │                                    │      │

│  └────────────────────────────────────┘      │
│  ...                                         │
│                                              │
│  [もっと見る]                                 │
│                                              │
└──────────────────────────────────────────────┘
```

### ユーザー操作

| 操作 | 挙動 |
|------|------|
| 「戻る」押下 | ブラウザバック |
| 「聴いたにする」押下 | 聴取記録を追加、ボタンが「聴いた」に変化 |
| 「聴いた」押下 | 聴取記録を削除、ボタンが「聴いたにする」に変化 |
| 星をクリック | 評価を選択（1〜5） |
| 星にホバー | ホバー中の星までハイライト |
| 「投稿する」押下 | レビューを作成、成功メッセージ表示後にレビュー一覧を更新 |
| 「配信元で聴く」押下 | 外部リンクを新規タブで開く |
| レビュー内のユーザー名押下 | `/users/[username]` へ遷移 |
| 「もっと見る」押下 | 次のページのレビューを読み込み |

### バリデーション

- 評価: 必須（1〜5）
- コメント: 任意、最大1000文字

### 状態別表示

| 状態 | 表示 |
|------|------|
| ローディング | Loading コンポーネント表示 |
| エピソード未発見 | 「エピソードが見つかりませんでした」 |
| 聴取記録トグル中 | ボタンにスピナー表示 |
| レビュー投稿成功 | 「レビューを投稿しました」メッセージ表示 |
| レビュー 0 件 | 「まだレビューはありません」 |
| エラー | ErrorMessage コンポーネント表示 |

### 使用 API

- `GET /api/v1/episodes/{id}`
- `POST /api/v1/episodes/{id}/listen`
- `DELETE /api/v1/episodes/{id}/listen`
- `GET /api/v1/episodes/{id}/listen`
- `POST /api/v1/episodes/{id}/reviews`
- `GET /api/v1/episodes/{id}/reviews?limit=10&offset=0`

## 認証フロー

```
未ログイン
  │
  ├─→ / (ランディング) ─→ /signup ─→ Supabase認証 ─→ /dashboard
  │                      /login  ─→ Supabase認証 ─→ /dashboard
  │
  └─→ /dashboard 等にアクセス ─→ /login へリダイレクト

ログイン済み・プロフィール未設定
  │
  └─→ /dashboard 等にアクセス ─→ /profile/setup へリダイレクト

ログイン済み・プロフィール設定済み
  │
  └─→ 全 (main) ページにアクセス可能
```
