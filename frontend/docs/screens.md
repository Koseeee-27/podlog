# 画面仕様書

## 概要

PodLog フロントエンドの全画面仕様を定義する。各画面について目的、URL、表示要素、ユーザー操作、認証要否を記載する。

本ドキュメントは `docs/requirements.md`（機能要件書）に基づいて作成されている。

### 設計方針

- **フルオープン型**: 閲覧系の画面はすべて認証不要。書き込み操作のみログインを必須とする
- **Google ソーシャルログイン**: メール+パスワード認証は MVP では提供しない
- **アプリ内 DB 検索**: 番組情報はアプリ側で管理し、ユーザーはアプリ内の DB を検索する

## 共通コンポーネント: Navbar

ログイン状態によって右側の表示を切り替える。

### デスクトップ（未ログイン時）

```
┌──────────────────────────────────────────────────┐
│  PodLog              検索              [ログイン]  │
└──────────────────────────────────────────────────┘
```

### デスクトップ（ログイン済み時）

```
┌──────────────────────────────────────────────────┐
│  PodLog              検索          [Avatar] 名前  │
└──────────────────────────────────────────────────┘
```

### モバイル（共通）

```
┌──────────────────────────────────────────────────┐
│  PodLog                                    [≡]   │
└──────────────────────────────────────────────────┘
```

ハンバーガーメニュー押下でサイドバーが開く。

**未ログイン時のメニュー:**

```
┌────────────────────┐
│                     │
│  検索               │  → /search
│                     │
│  ログイン           │  → /login
│                     │
└────────────────────┘
```

**ログイン済み時のメニュー:**

```
┌────────────────────┐
│  [Avatar]           │
│  表示名             │
│  @username          │
│                     │
│  検索               │  → /search
│  マイページ          │  → /users/{username}
│  設定               │  → /settings
│                     │

```

### Navbar の操作

| 要素 | 挙動 |
|------|------|
| ロゴ（PodLog） | `/` へ遷移 |
| 検索 | `/search` へ遷移 |
| ログインボタン（未ログイン時） | `/login` へ遷移 |
| アバター+名前（ログイン済み時） | `/users/{username}` へ遷移 |

## 2. ログイン

- **URL**: `/login`
- **認証**: 不要
- **目的**: Google アカウントでのログイン / 新規登録

MVP では Google ソーシャルログインのみ提供する。新規登録とログインを同一画面で扱う（Google 認証ではアカウントが存在しなければ自動作成される）。

### 表示要素

```
┌──────────────────────────────────────────────────┐
│                                                  │
│              PodLog にログイン                     │
│                                                  │
│       [G  Google アカウントでログイン]              │
│                                                  │
│       ログインすることで利用規約と                  │
│       プライバシーポリシーに同意します               │
│                                                  │
└──────────────────────────────────────────────────┘
```

### ユーザー操作

| 操作 | 挙動 |
|------|------|
| 「Google アカウントでログイン」押下 | Google OAuth 認証フローを開始。認証成功後、プロフィール未設定なら `/profile/setup` へ、設定済みなら `/` へ遷移 |

### 状態別表示

| 状態 | 表示 |
|------|------|
| 認証処理中 | ボタンが無効化され、ローディング表示 |
| エラー | ボタン下部にエラーメッセージ表示 |
| ログイン済みでアクセス | `/` へリダイレクト |

## 3. プロフィール初期設定

- **URL**: `/profile/setup`
- **認証**: 必要
- **目的**: 新規登録後のプロフィール初期設定（ユーザー名・表示名の必須設定）

### 表示要素

```
┌──────────────────────────────────────────────────┐
│  [Navbar]                                        │
├──────────────────────────────────────────────────┤
│                                                  │
│           プロフィール設定                         │
│   はじめに、プロフィールを設定しましょう             │
│                                                  │
│  ユーザー名:   [________________]                 │
│  （英数字とアンダースコア、3〜30文字）               │
│  （あとから変更できません）                         │
│                                                  │
│  表示名:       [________________]                 │
│                                                  │
│              [設定を完了する]                      │
│                                                  │
└──────────────────────────────────────────────────┘
```

### ユーザー操作

| 操作 | 挙動 |
|------|------|
| 「設定を完了する」押下 | プロフィール作成 → `/` へ遷移 |

### バリデーション

- ユーザー名: 必須、3〜30文字、英数字とアンダースコアのみ、一意であること
- 表示名: 必須

### 状態別表示

| 状態 | 表示 |
|------|------|
| 未認証 | `/login` へリダイレクト |
| プロフィール設定済み | `/` へリダイレクト |
| ユーザー名が既に使用されている | フィールド下にエラーメッセージ |
| エラー | フォーム上部にエラーメッセージ表示 |
| 送信中 | ボタンが無効化される |

- **認証**: 不要
- **目的**: アプリ内 DB に登録されている番組をキーワードで検索する

### 検索結果カードの表示項目

- 番組アートワーク
- 番組タイトル
- 配信者名
- 平均評価、レビュー件数

### ユーザー操作

| 操作 | 挙動 |
|------|------|
| 検索入力 | デバウンス後にアプリ内 DB を検索 |
| 番組カード押下 | `/podcasts/[id]` へ遷移 |
| 「番組の追加をリクエストする」押下 | 未ログインなら `/login` へ遷移。ログイン済みなら追加リクエストダイアログを表示 |

### 番組追加リクエストダイアログ

ログイン済みユーザーが検索で番組を見つけられなかった場合に、追加をリクエストする。

```
┌──────────────────────────────────────────┐
│                                          │
│  番組の追加をリクエスト                    │
│                                          │
│  番組名:     [________________]          │
│  番組URL:    [________________]          │
│  （Apple Podcasts や Spotify のURL）      │
│                                          │
│        [キャンセル]  [リクエストする]      │
│                                          │
└──────────────────────────────────────────┘
```

### 状態別表示

| 状態 | 表示 |
|------|------|
| 初期（未検索） | 「番組名を入力して検索してください」 |
| ローディング | 検索バーにスピナー表示 |
| 結果あり | 番組カードの一覧 |
| 結果なし | 「検索結果が見つかりませんでした」+ 追加リクエスト導線 |
| エラー | エラーメッセージ表示 |

- 「聴いたにする」ボタンの代わりに「ログインして記録する」を表示
- レビュー投稿フォームの代わりに「ログインしてレビューを書く」を表示
- レビュー一覧は閲覧可能

### 自分のレビューが既にある場合

レビュー投稿フォームの代わりに、自分のレビューを編集可能な状態で表示する。

```
┌──────────────────────────────────────────┐
│ あなたのレビュー                          │

```

| 操作 | 挙動 |
|------|------|
| 「戻る」押下 | ブラウザバック |
| 番組名押下 | `/podcasts/[id]` へ遷移 |
| 「聴いたにする」押下 | 聴取記録を追加、ボタンが「聴いた」に変化 |
| 「聴いた」押下 | 聴取記録を取り消し、ボタンが「聴いたにする」に変化 |
| 星をクリック | 評価を選択（1〜5） |
| 「投稿する」押下 | レビューを作成、レビュー一覧を更新 |
| 「編集する」押下 | 自分のレビューを編集モードに切り替え |
| 「削除する」押下 | 確認ダイアログ表示後、レビューを削除 |
| レビュー内のユーザー名 / アバター押下 | `/users/[username]` へ遷移 |
| 「もっと見る」押下 | 次のページのレビューを読み込み |
| 「ログインして記録する」押下（未ログイン時） | `/login` へ遷移 |
| 「ログインしてレビューを書く」押下（未ログイン時） | `/login` へ遷移 |

### バリデーション

- 評価: 必須（1〜5の整数）
- コメント: 任意、最大1000文字

### 状態別表示

| 状態 | 表示 |

| レビュー 0 件 | 「まだレビューはありません。最初のレビューを書きましょう！」 |
| エラー | エラーメッセージ表示 |

## 8. プロフィール編集

- **URL**: `/settings/profile`
- **認証**: 必要
- **目的**: 表示名・アバター・自己紹介・好きな番組の編集

### 表示要素

```
┌──────────────────────────────────────────────────┐
│  [Navbar]                                        │
├──────────────────────────────────────────────────┤
│                                                  │

│  └────────┘                                      │
│                                                  │
│  表示名:       [________________]                 │
│                                                  │
│  自己紹介:     [________________]                 │
│               [________________]                 │
│                                                  │
│  ── 好きな番組 ─────────────────────────────      │
│                                                  │
│  ┌────────┐ ┌────────┐ ┌────────┐              │
│  │[Artwork]│ │[Artwork]│ │  ＋    │              │
│  │ 番組名  │ │ 番組名  │ │ 追加   │              │
│  │   [×]  │ │   [×]  │ │        │              │
│  └────────┘ └────────┘ └────────┘              │
│                                                  │
│         [キャンセル]  [保存する]                    │
│                                                  │
└──────────────────────────────────────────────────┘
```

### アバター画像のアップロード

- 「画像を変更する」を押すとファイル選択ダイアログが開く
- 対応形式: JPEG, PNG
- プレビューが即座に表示される

### 好きな番組の編集

- 「＋ 追加」を押すと番組検索ダイアログが開く
- 番組名で検索し、結果から選択して追加する
- 既に追加済みの番組の「×」ボタンで削除

### ユーザー操作

| 操作 | 挙動 |
|------|------|
| 「画像を変更する」押下 | ファイル選択ダイアログを表示 |
| 「＋ 追加」押下 | 番組検索ダイアログを表示 |
| 番組の「×」押下 | 好きな番組リストから削除 |
| 「保存する」押下 | プロフィール更新 → `/users/{username}` へ遷移 |
| 「キャンセル」押下 | 変更を破棄して前の画面に戻る |

### バリデーション

- 表示名: 必須
- アバター画像: JPEG / PNG、ファイルサイズ上限あり

### 状態別表示

| 状態 | 表示 |
|------|------|
| ローディング | ローディング表示 |
| 未認証 | `/login` へリダイレクト |
| 画像アップロード中 | アバターエリアにスピナー表示 |
| 保存成功 | ユーザーページへ遷移 |
| エラー | フォーム上部にエラーメッセージ表示 |

## 認証フロー

```
未ログイン
  │
  ├─→ /（トップ）→ 「Google で始める」→ Google 認証
  │                                        │
  │                                        ├─→ 初回 → /profile/setup → / 
  │                                        └─→ 2回目以降 → /
  │
  ├─→ /login → Google 認証 → 同上
  │
  ├─→ /search, /podcasts/[id], /episodes/[id], /users/[username]
  │     → 認証なしで閲覧可能
  │     → 書き込み操作時に /login へ誘導
  │
  └─→ /settings, /settings/profile にアクセス → /login へリダイレクト

ログイン済み・プロフィール未設定
  │
  └─→ 書き込み操作時 → /profile/setup へリダイレクト

ログイン済み・プロフィール設定済み
  │
  └─→ 全ページにアクセス可能、全操作が利用可能
```
